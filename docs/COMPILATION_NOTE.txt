LaTeX Compilation Note
=====================

This workspace does not have LaTeX installed (pdflatex, bibtex, makeindex not found).

To compile the report, you have two options:

OPTION 1: Install LaTeX on Pop!_OS
-----------------------------------
sudo apt-get update
sudo apt-get install texlive-full

Then compile with:
cd /home/dio/ros2_ws/docs
./compile_report.sh

OPTION 2: Use Overleaf (Online)
--------------------------------
1. Create account at https://www.overleaf.com
2. Create new project (Upload Project)
3. Upload all files from docs/ directory:
   - final_project_report.tex
   - preamble.tex
   - commands.tex
   - references.bib
   - screenshots/ folder (entire directory)
4. Set final_project_report.tex as main document
5. Compile online
